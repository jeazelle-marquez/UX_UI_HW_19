my UX skills

user research

to get the most information about a user, it’s crucial to start with secondary research first before getting into primary research. a great foundation of user research leads to a great product.

qualitative interviews

for every interview, I try my best to make the interviewee feel comfortable. making a safe and comfortable interview environment for users is key to generating open and honest answers.

user empathy

putting yourself in the shoes of the user is the number one skill any UX designer needs in order to be successful. having this skill allows me to think and create solutions for problems as a user would.
